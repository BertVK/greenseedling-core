# GreenSeedling Core

**Modular indoor grow box controller** developed and documented by [GreenSeedling.eu](https://greenseedling.eu).

GreenSeedling Core provides a fully documented platform for DIY and professional indoor plant cultivation.  
This repository contains:

- Hardware schematics, BOM, and build guides
- Firmware and software for controlling the grow box
- Documentation for Scope 1 features and future upgrades
